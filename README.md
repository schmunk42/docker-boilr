# boilr

:octocat: https://github.com/tmrts/boilr :whale: https://hub.docker.com/r/schmunk42/boilr/

## Installation

In your `.bashrc` add

    alias "boilr=docker run -it --rm -v ~/.config/boilr:/root/.config/boilr -v $(pwd):/project --workdir /project schmunk42/boilr"
